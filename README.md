# Restaurant List
A simple web application for restaurant finding

## Features
- listing restaurants
- searching restaurants by name
- details for the restaurant you choose


### Searching
type the name of the restaurants in search bar

### Built with
- fortawesome: 5.14.0
- popper.js: 2.5.2
- express.js: 4.17.1
- express-handlebars.js: 5.1.0
- nodemon.js: 2.0.4
